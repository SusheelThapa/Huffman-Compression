# Huffman-Compression

## Setting up Project

### Linux

1. First of all give `install.sh` file executable permission

   ```sh
   chmod +x install.sh
   ```

2. Run `install.sh`

   ```sh
   ./install.sh
   ```

**_You can head over running the project in [Linux](#linux-1)_**

## Running the Project

### Linux

1. Compiling the project

   ```sh
   make all
   ```

2. Running project

   ```sh
   ./main
   ```
